# Miller Metals — Custom Metal Fabrication Website

A modern, responsive business website for **Siller Metals**, a professional metal fabrication company specializing in custom metalwork, shipping container homes, and structural builds. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Dark / Light mode** toggle with persistent theme context
- **Smooth animations** powered by Framer Motion
- **Fully responsive** layout for mobile, tablet, and desktop
- **Contact form** with EmailJS integration

---

## Sections

| Section               | Description                                               |
| --------------------- | --------------------------------------------------------- |
| **Navbar**            | Sticky navigation with dark/light mode toggle             |
| **Hero**              | Full-screen landing with headline, stats, and CTA buttons |
| **About**             | Company background and owner profile                      |
| **Services**          | Cards showcasing the five core service offerings          |
| **Portfolio**         | Gallery of completed projects                             |
| **Featured Projects** | Highlighted showcase pieces                               |
| **Before & After**    | Side-by-side comparison slider for restoration work       |
| **Testimonials**      | Customer reviews and social proof                         |
| **FAQ**               | Frequently asked questions accordion                      |
| **Contact**           | Quote request form                                        |
| **Footer**            | Links, contact info, and social media                     |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
